Spanish:

Descargar el archivo comprimido desde esta web.

Descomprimir el archivo .zip en el directorio (/wp-content/plugins/) Sin modificarlo sino podrias tener problemas.

Una ves echo esto tienes que activar el security captcha

Ahora podras fijarte en cada FORM de comentarios que el plugin funciona ;)
